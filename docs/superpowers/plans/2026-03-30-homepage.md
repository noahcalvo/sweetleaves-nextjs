# Homepage Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the WordPress-driven homepage with a static Figma-designed homepage featuring hero, product grid, deals banner, store photos, review carousel, blog carousel, Garden Club promo, and FAQ section.

**Architecture:** Server components by default, `"use client"` only for carousels (ReviewsSection, BlogSection) and FAQ accordion (HomeFaqSection). All old WordPress-based home components will be deleted. Static content replaces dynamic WordPress data.

**Tech Stack:** Next.js 15 App Router, Tailwind CSS v4 with project tokens (`dark-green`, `dark-sage`, `sage`, `parchment`, `light-gold`, `orange-glow`, `sky-blue`, `ivory`, `almost-black`), Poppins font utilities (`font-poppins-regular`, `font-poppins-semibold`, `font-poppins-bold`), `text-display` (45px) size token.

// app/components/home/ReviewsSection.tsx
"use client";

import { useState } from "react";

const REVIEWS = [
  {
    text: "This is the place for your cannabis needs. I love the variety of options! The employees are super knowledgeable and helped me find the perfect product for me!",
    author: "Katie W.",
  },
  {
    text: "Beautiful store and best experience. Great product selection and even better customer service! The employees are knowledgeable and answered all my questions. 10/10 will be back!",
    author: "Ella T.",
  },
  {
    text: "This is the spot. If you enjoy great customer service and knowledgeable staff, you're in luck. Great assortment and impressive store layout. 10/10.",
    author: "Luis N.",
  },
];

const GOOGLE_REVIEW_URL =
  "https://search.google.com/local/writereview?placeid=ChIJrQjHvHYzs1IRgEbpaU30vFE";

function CarouselArrow({
  direction,
  onClick,
}: {
  direction: "left" | "right";
  onClick: () => void;
}) {
  return (
    <button
      onClick={onClick}
      className={`bg-orange-glow rounded-full size-[68px] flex items-center justify-center shrink-0 ${
        direction === "left" ? "rotate-180" : ""
      }`}
      aria-label={direction === "left" ? "Previous review" : "Next review"}
    >
      <span className="font-poppins-semibold text-[55px] text-white leading-none">
        &gt;
      </span>
    </button>
  );
}

export default function ReviewsSection() {
  const [activeIndex, setActiveIndex] = useState(0);

  const prev = () =>
    setActiveIndex((i) => (i - 1 + REVIEWS.length) % REVIEWS.length);
  const next = () =>
    setActiveIndex((i) => (i + 1) % REVIEWS.length);

  const review = REVIEWS[activeIndex];

  return (
    <section className="bg-parchment border border-sage rounded-[30px] md:rounded-[40px] flex flex-col gap-[25px] md:gap-[25px] items-center justify-center px-7 md:px-[30px] py-10 md:py-[45px] overflow-hidden flex-1">
      <h2 className="font-poppins-bold text-[30px] md:text-display text-orange-glow text-center w-full">
        From the Community
      </h2>

      <a
        href={GOOGLE_REVIEW_URL}
        target="_blank"
        rel="noopener noreferrer"
        className="bg-white rounded-[20px] px-8 md:px-9 py-8 md:py-9 w-full max-w-[608px] text-left"
      >
        <p className="font-poppins text-[28px] md:text-[38px] text-orange-glow italic leading-normal">
          ★★★★★
        </p>
        <p className="font-poppins text-[16px] md:text-[18px] text-almost-black italic leading-normal mt-2.5">
          {review.text}
        </p>
        <p className="font-poppins-bold text-[16px] md:text-[18px] text-almost-black mt-2.5">
          - {review.author}
        </p>
      </a>

      <div className="flex gap-[10px] md:gap-[25px] items-center justify-center w-full">
        <CarouselArrow direction="left" onClick={prev} />
        <a
          href={GOOGLE_REVIEW_URL}
          target="_blank"
          rel="noopener noreferrer"
          className="bg-light-gold text-dark-green font-poppins-semibold uppercase text-base px-6 py-5 rounded-full hover:opacity-90 transition-opacity text-center flex-1 max-w-[404px]"
